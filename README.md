# 図書館検索サービス

Calil API を使用して、日本全国の図書館を簡単に検索できるシンプルなウェブアプリケーションです。

## 機能

- **キーワード検索**: 図書館名または地名でキーワード検索
- **詳細情報表示**: 図書館システム、都道府県、住所などの情報を表示
- **シンプルなUI**: 使いやすいミニマル設計

## 使い方

1. `index.html` をブラウザで開く
2. キーワード（例: 「東京」「中央図書館」）を入力
3. 「検索」ボタンをクリックまたはEnterキーを押す
4. 検索結果に該当する図書館が表示されます

## 技術スタック

- HTML5
- CSS3 (レスポンシブデザイン)
- Vanilla JavaScript (フレームワークなし)
- Calil API (図書館検索API)

## API について

このアプリケーションは [Calil API](https://calil.jp/api/) を使用しています。

- **エンドポイント**: `https://api.calil.jp/library/search.php`
- **パラメータ**:
  - `appkey`: APIキー
  - `keyword`: 検索キーワード
  - `format`: レスポンス形式（json）

## セットアップ

1. リポジトリをクローン
2. `index.html` をブラウザで開く
3. すぐに使用可能（追加インストール不要）
